## Problem

Headline:

```text
1画面では、情報が“順番待ち”になります。
```

Copy:

```text
案内、キャンペーン、QRが時間で切り替わると、来館者がQRを見る前に通り過ぎたり、重要な案内が表示される前に見落とされたりします。
```

Comparison:

| 1画面 | Misell 3連 |
| --- | --- |
| 案内→キャンペーン→QRが順番に切り替わる | 左:案内、中央:訴求、右:QRを同時表示 |
| QRが出る前に通り過ぎる | QRを常時表示できる |
| 動画とQRが画面を奪い合う | 中央で見せて右で誘導できる |
| 効果が見えにくい | QR反応を確認できる |

## Solution

```text
Misellは、施設内の見られている場所を3連メディアに変えます。
```

Features:

- 3連表示 / 3面ワイド表示
- 顧客セルフ更新
- QR計測
- 端末監視
- 業種別テンプレート
- 将来の広告枠化
