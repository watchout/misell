# 05. 60 Second Demo Video Script

## Timeline

| Time | Visual | Telop | Narration |
| --- | --- | --- | --- |
| 0-5s | Empty wall / poster / single screen | 施設内の“見られる場所”が活かされていない | 施設には、来館者が必ず見る場所があります。 |
| 5-12s | Single screen rotates guide/campaign/QR | 1画面では情報が順番待ち | 1画面では、案内、キャンペーン、QRが順番待ちになります。 |
| 12-22s | 3 monitors turn on | 左:案内 / 中央:訴求 / 右:QR | Misellは、その場所を3連の訴求面に変えます。 |
| 22-32s | Admin/Studio style update | 自分で更新 / QR計測 / 端末監視 | 30日以内に、自社更新、QR計測、3連表示ができる状態を立ち上げます。 |
| 32-42s | Hotel, sauna, karaoke, meeting room mockups | 業種別に使える | ホテル、温浴、カラオケ、貸会議室など、施設ごとの見せ方に合わせます。 |
| 42-52s | QR reaction/report screen | 放映ログとQR反応を確認 | 売上を保証するのではなく、放映と反応を確認しながら改善します。 |
| 52-60s | Offer screen | 初期150,000円 / 月額19,800円 / 税別 | 初期導入パートナーは、設置場所診断からご相談ください。 |

## Export Direction

- 16:9 mp4 for sales meeting.
- Keep text large.
- Avoid long body copy.
- Use only approved price/guarantee text.

## Forbidden

- 売上増加保証
- 広告収益保証
- QR数保証
- ROAS保証
- 日本初/唯一
