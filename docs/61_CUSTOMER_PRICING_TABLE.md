# 61. 顧客提示用価格表

## 目的

Misellの初回商談、見積、LP、申込書で使える顧客提示用の価格表を定義する。

価格は正式見積前の営業目安であり、標準施工範囲、端末構成、設置難易度、運用範囲により調整する。

## 価格方針

- 端末販売ではなく、施設内メディア運用として販売する
- 初期費用は導入障壁を下げるため、標準範囲を明確にして提示する
- 月額には更新、監視、軽微な修正、レポートを含める
- 広告収益は保証せず、Mediaプランで広告運用支援とレポートを提供する
- AI Edgeは本導入後の上位拡張として扱い、初回テスト導入では過剰に約束しない

| 月額費用 | 49,800円から | 79,800円から | 128,000円から | Standard + 30,000円から |
| 推奨契約期間 | 12か月から | 24か月から | 24か月から | 24か月から |
| 表示方式 | 3連表示 | 3連表示 | 3連表示 | 3連表示 |
| 月次更新 | 月1回 | 月2回 | 月2回 + 広告更新 | 個別設計 |
| QR計測 | オプション | 標準 | 標準 | 標準 |
| 月次レポート | 簡易 | 標準 | 広告主向け含む | AIコメント含む |
| 広告枠運用 | なし | オプション | 標準 | 標準 |
| AI制作/分析 | なし | オプション | オプション | 標準/個別 |

## Lite

### 想定顧客

- 小規模施設
- まず案内/販促だけ試したい施設
- テスト導入後に低初期で継続したい施設

### 含まれる範囲

- 3画面連結表示
- LAN管理画面
- 基本playlist設定
- 月1回の素材差し替え
- 簡易死活確認
- 基本サポート

### 含まれない範囲

- 広告主対応
- 月次詳細レポート
- 大量素材制作
- 標準外施工
- AIカウント/AI分析

### 営業トーク

「まずは紙POPやポスター更新の負荷を減らし、館内案内と簡単な販促を動かすためのプランです。」

## Standard

### 想定顧客

- 本命の初回有償導入
- 案内、販促、QR計測を一体で運用したい施設
- 月次改善を前提にしたい施設

### 含まれる範囲

- Liteの範囲
- 月2回の素材差し替え
- QR計測
- 月次レポート
- POPタグ基本連動
- 運用改善提案

### 含まれない範囲

- 広告主営業代行
- 広告掲載審査の代行全般
- AIカウント/センサー
- 個別システム連携

### 営業トーク

「案内と販促を月次で回し、QRで反応を見ながら改善する標準プランです。最初の有償導入はStandardを基準にします。」

## Media

### 想定顧客

- 施設内広告枠を作りたい施設
- 地域広告主やスポンサーを取り込みたい施設
- 広告主向けレポートを出したい施設

### 含まれる範囲

- Standardの範囲
- 媒体資料作成
- 広告枠メニュー設計
- 広告素材の基本確認
- QR計測
- 広告主向け掲載レポート
- 広告運用改善提案

### 広告運用手数料

広告売上が発生する場合、広告売上の20%から40%を運用手数料の目安とする。

### 営業トーク

「施設内の見られる場所を、地域広告やスポンサー枠として使うプランです。初期はインプレッション保証ではなく、掲載枠、QR計測、月次レポートとして販売します。」

## AI Edge

### 想定顧客

- 複数施設展開後の上位顧客
- AI制作やAIレポートを使いたい施設
- 匿名カウントや多言語案内に進めたい施設

### 含まれる範囲

- StandardまたはMediaの範囲
- AI制作支援
- AIレポート文生成
- 匿名人数カウント、滞留推定、多言語案内などの個別機能

### 注意

AI Edgeは、設置場所、プライバシー掲示、カメラ有無、ネットワーク条件により設計が変わるため、初回商談では拡張余地として説明する。

## テスト導入価格

| メニュー | 期間 | 初期費用 | 月額/利用料 | 条件 |
| --- | --- | ---: | ---: | --- |
| デモ協力テスト | 2から4週間 | 実費または無償 | 無償から49,800円 | 事例化、写真/動画、フィードバック協力 |
| 有償PoC | 1から3か月 | 98,000円から298,000円 | 49,800円から79,800円 | 月次レポート、継続判断、設置場所固定 |
| 本導入前トライアル | 1か月 | 本導入初期費用に充当 | Standard相当 | 本契約移行を前提に条件調整 |

## 標準施工範囲

標準施工は以下を含む。

- 3画面設置
- 端末設置
- 既存電源利用
- HDMI/DP/USB-C等の標準配線
- LANまたは既存Wi-Fi接続
- 初期表示確認
- 端末キッティング
- 基本操作説明

## 標準外作業

以下は追加見積とする。

| 作業 | 価格目安 |
| --- | ---: |
| 新規電源工事 | 個別見積 |
| 天井/壁面下地補強 | 個別見積 |
| 夜間/閉店後施工 | 50,000円から |
| 長距離配線 | 個別見積 |
| 特注金具 | 実費 + 管理費 |
| 既存設備撤去 | 個別見積 |
| 大量素材制作 | 30,000円から/本 |
| 個別システム連携 | 300,000円から |

## 追加オプション

| オプション | 価格目安 | 内容 |
| --- | ---: | --- |
| 追加素材制作 | 30,000円から/本 | 15秒から30秒の広告/案内素材 |
| 月次更新追加 | 20,000円から/月 | 標準回数を超える更新 |
| QR/LP追加 | 10,000円から/本 | キャンペーンQR、簡易LP |
| 月次レポート強化 | 20,000円から/月 | 広告主別/時間帯別の詳細化 |
| 現地対応 | 30,000円から/回 | 遠隔で解決できない現地作業 |

## 見積前の確認事項

- 設置画面の台数、インチ、所有者
- 端末の有無
- 電源位置
- LAN/Wi-Fi条件
- 画面設置方法
- 施工可能時間
- 表示したい内容
- QR誘導先
- 広告掲載可否
- 事例掲載可否
- 希望契約期間
- 標準外作業の有無

## 顧客提示時の注意

- 広告収益、QR数、視認者数は保証しない
- AIカウントはプライバシー掲示と設置条件の確認後に提案する
- 標準外施工は必ず別見積にする
- テスト導入で無償にする場合も、写真/動画/事例化/フィードバック条件を明記する
- 初期費用を過度に下げる場合は最低利用期間を設定する
