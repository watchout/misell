# 01. 2027年6月 10億円バイアウト戦略

## 目標

2027年6月までに、連結サイネージ「ミセル」を **10億円で買収検討される事業** に育てる。

この目標は、単発の機材販売・施工業では達成しにくい。評価されるのは、継続課金、導入ネットワーク、広告運用データ、再現性ある販売・施工・運用プロセスである。

## 買収される事業の条件

買い手が評価するものは以下。

1. MRR（月次継続売上）
2. 導入店舗数
3. 解約率の低さ
4. 広告掲載率
5. 1店舗あたりの採算性
6. 標準化された施工・運用手順
7. 導入先ネットワーク
8. 地域広告主ネットワーク
9. 放映ログ・QRログ・広告効果レポート
10. AI/データ活用による拡張余地

## 数値目標

10億円バイアウトを狙うための目安。

### Base Case

- 導入店舗数：100店舗
- 平均月額売上：80,000円
- MRR：800万円
- ARR：9,600万円
- 評価倍率：ARR 8〜12倍
- 想定評価：7.7億〜11.5億円

### Growth Case

- 導入店舗数：150店舗
- 平均月額売上：100,000円
- MRR：1,500万円
- ARR：1.8億円
- 評価倍率：ARR 6〜10倍
- 想定評価：10.8億〜18億円

### Media Case

- 導入店舗数：100店舗規模
- 店舗月額と広告運用を組み合わせる
- MRRとARRは承認済みのpricing dataと広告運用実績から算出する
- メディア性が評価されれば10億円レンジを狙える

## 2027年6月までのロードマップ

### 2026年6月〜8月：MVP検証

- NiPoGi端末でLinux版MVP構築
- 3連表示、LAN素材更新、タイマー配信を実装
- 安心お宿/バリアン/パセラ/ビジョンセンターへデモ
- 最初のテスト導入1〜3件
- 導入写真、デモ動画、運用ログを取得

### 2026年9月〜12月：有償PoC化

- 月額運用プラン開始
- 初期費用を抑えた36か月契約プランを整備
- 10店舗導入を目標
- POPタグ/QR計測を追加
- 月次レポートを手動+半自動で提供

### 2027年1月〜3月：店舗メディア化

- 30店舗導入を目標
- 広告主向け媒体資料作成
- 地域広告枠の販売開始
- 広告主別レポート作成
- AIによるコンテンツ生成、月次改善提案を実装

### 2027年4月〜6月：バイアウト準備

- 50〜100店舗導入を目標
- MRR 500万〜1,000万円を目指す
- 買収候補向け事業資料を作成
- KPI、契約、導入先、広告主、採算構造を整理
- 提携/資本業務提携/売却交渉を開始

## 買い手候補

- デジタルサイネージ会社
- LEDビジョン/映像設備会社
- 広告代理店
- 地域広告会社
- 店舗DX会社
- POS/レジ会社
- 電子棚札/小売IoT会社
- ホテル/観光DX会社
- 通信工事会社
- 施設運営会社

## 買い手に刺さるストーリー

「ミセルは、単なる3連サイネージではなく、中小店舗・施設の空きスペースを地域広告メディアに変えるネットワークである。施工、端末、CMS、広告運用、効果レポート、AI改善まで標準化し、買い手の既存顧客基盤に横展開できる。」

## 重要KPI

| KPI | 初期目標 | 2027年6月目標 |
|---|---:|---:|
| 導入店舗数 | 3 | 50〜100 |
| MRR | 30万円 | 500万〜1,000万円 |
| 平均月額売上 | 8万円 | 8万〜12万円 |
| 広告掲載率 | 10% | 50%以上 |
| 1店舗採算性 | private storeで管理 | private storeで管理 |
| 解約率 | 未計測 | 月2%以下 |
| 導入標準工期 | 1〜2日 | 1日以内 |
| 月次運用工数 | 手動 | 1店舗2時間以内 |

## してはいけないこと

- ただのモニター販売に寄せる
- 案件ごとにフルカスタムする
- 初期費用だけで利益を取りにいく
- 月額価値を作らない
- 導入後のデータを取らない
- 広告運用を後回しにしすぎる
- 施工仕様を標準化しない

## 方針

初期は小さく作る。ただし、最初から買収される事業としてKPI・契約・データ・標準仕様を設計する。
