# 62. Misell Media 媒体資料

## 目的

広告主または広告販売パートナーへ提示できる、Misell Mediaの媒体資料を定義する。

Misell Mediaは、ホテル、温浴施設、貸会議室、カラオケ/飲食、観光施設などの施設内3連サイネージへ広告を掲載できる地域密着型メディアである。

## 媒体概要

| 項目 | 内容 |
| --- | --- |
| 媒体名 | Misell Media |
| 掲載場所 | 施設内3連サイネージ |
| 表示形式 | 3面wide、中央広告 + QR、3分割広告、館内案内連動 |
| 掲載単位 | 1週間、1か月 |
| 計測 | 放映回数、QR読み取り数、時間帯別アクセス |
| レポート | 月次簡易レポート、広告主別レポート |

## 媒体の特徴

- 施設内に滞在している来館者へ広告を届けられる
- 館内案内や施設サービスと同じ導線に広告枠を作れる
- QRでクーポン、予約、資料請求、LPへ誘導できる
- 掲載後に放映回数とQR反応をレポートできる
- 地域広告、求人、観光、イベント協賛と相性が良い

## 掲載施設カテゴリ

| カテゴリ | 想定設置場所 | 主な視認者 | 相性の良い広告 |
| --- | --- | --- | --- |
| ホテル/宿泊 | フロント、ロビー、朝食会場 | 宿泊者、訪日客、周辺利用者 | 飲食、観光、交通、土産、予約導線 |
| 温浴/休憩施設 | 受付、休憩所、飲食エリア | 長時間滞在者、リピーター | 飲食、美容、健康、周辺店舗 |
| カラオケ/飲食 | 受付、店内導線、イベントスペース | グループ客、若年層、イベント参加者 | フード、コラボ、イベント、求人 |
| 貸会議室 | 受付、会議室前、EVホール | 企業担当者、登壇者、来場者 | BtoBサービス、配信、採用、スポンサー |
| 観光/地域施設 | 案内所、休憩所、物販周辺 | 観光客、地域利用者 | 観光体験、飲食、交通、自治体施策 |

## QR計測

広告ごとに専用QRを発行し、LP、予約ページ、クーポン、問い合わせフォームへ誘導する。

### 計測項目

- QR読み取り数
- 日別アクセス
- 時間帯別アクセス
- 掲載施設別アクセス
- 広告素材別アクセス

### レポート例

| 項目 | 内容 |
| --- | --- |
| 掲載期間 | 2026年7月1日から7月31日 |
| 放映回数 | 4,320回想定 |
| QR読み取り数 | 126 |
| 反応が多い時間帯 | 18時から21時 |
| 改善提案 | 夜帯向けコピー、クーポン訴求、右画面QR強調 |

## 掲載審査

掲載前に施設ブランドと法令に照らして確認する。

### 掲載不可または要確認

- 虚偽、誇大広告
- 法令違反のおそれがある広告
- 著作権、肖像権の確認が取れていない素材
- 施設ブランドを損なう表現
- 公序良俗に反する内容
- 医療、金融、投資、求人など審査が必要な表現
- 施設側の競合または不適切と判断される業種

## 申込フロー

1. 掲載希望施設、期間、広告枠を選択
2. 素材または制作依頼内容を提出
3. 掲載審査
4. QR/LP設定
5. 掲載スケジュール確定
6. 掲載開始
7. 月次レポート提出
8. 次月継続または改善提案

## 広告主向け営業トーク

「Misell Mediaは、施設内で実際に滞在しているお客様に広告を届ける地域密着型メディアです。Web広告や紙チラシだけでは届きにくい現場の導線に、QR計測付きで広告を出せます。」

## 施設向け営業トーク

「施設内の案内画面を、地域広告やスポンサー枠として活用できます。最初は広告収益を保証せず、掲載枠、QR計測、月次レポートとして小さく始めます。」

## 初期販売時の注意

- インプレッション保証はしない
- 視認者数保証はしない
- 属性推定やAIカウントは上位プラン導入後に扱う
- 施設別の掲載可否とNG業種を必ず確認する
- パートナー条件はpublic repositoryに置かず、private storeで事前承認する

## 提案時に確認すること

- 掲載希望施設
- 掲載期間
- 広告素材の有無
- QR誘導先
- 掲載目的
- 予算
- NG業種
- レポートで見たい項目
- 継続掲載の判断基準
