# 63. テスト導入 簡易合意書テンプレート

## 目的

Misellのテスト導入時に、口約束で始めず、期間、費用、設置物、事例掲載、データ利用、撤去条件を事前に確認するための簡易合意書テンプレートを定義する。

この文書は営業・運用のたたき台であり、正式な契約書ではない。顧客提示前に、案件規模、設置リスク、個人情報の取扱い、広告掲載有無に応じて法務確認を行う。

## 簡易合意書本文

以下、`甲` は導入先施設または運営会社、`乙` はMisell提供者を指す。

### 第1条 目的

甲および乙は、甲の施設内における連結サイネージ「Misell」の有用性を検証するため、以下の条件によりテスト導入を実施する。

本テスト導入は、施設内での視認性、案内・販促効果、QR導線、運用負荷、継続利用可能性を確認することを目的とし、広告収益、来館者数増加、売上増加その他特定の成果を保証するものではない。

### 第2条 テスト導入期間

1. テスト導入期間は、開始日から終了日までの期間とする。
2. 標準期間は2週間から4週間とする。
3. 期間延長を行う場合は、終了日の3営業日前までに甲乙協議のうえ、延長期間、費用、撤去予定日を確認する。

**記入欄**

| 項目 | 内容 |
| --- | --- |
| 開始日 | 20XX年XX月XX日 |
| 終了日 | 20XX年XX月XX日 |
| 設置予定日 | 20XX年XX月XX日 |
| 撤去予定日 | 20XX年XX月XX日 |
| 延長可否 | 可 / 不可 / 協議 |

### 第3条 設置場所・機材

1. 設置場所は、甲乙が事前に確認した施設内の指定場所とする。
2. 乙は、テスト導入に必要な端末、画面、配線、表示ソフトウェア、初期コンテンツを準備する。ただし、別途合意した場合を除く。
3. 甲は、設置に必要な電源、設置スペース、通信環境、作業可能時間を提供する。
4. 標準外工事、新規電源工事、高所作業、下地補強、夜間作業、特注金具、既存設備撤去が必要な場合は、別途見積とする。

### 第4条 費用

1. テスト導入費用は、以下の記入欄に定める。
2. 無償または低額で実施する場合でも、事例化、写真/動画撮影、フィードバック、レポート作成への協力条件を明記する。
3. 標準外作業、追加素材制作、現地再訪、撤去延期、破損対応は別途見積とする。

| 追加素材制作 | 含む / 別途 |

### 第5条 表示内容・素材

1. 表示内容は、甲乙が合意した案内、販促、広告枠サンプル、QR導線、その他テスト目的に必要な内容とする。
2. 甲が提供するロゴ、写真、動画、文章、URLその他素材について、甲は乙が本テスト導入の範囲で利用する権限を有することを確認する。
3. 第三者の権利を侵害するおそれのある素材、法令違反、公序良俗違反、施設ブランドを損なう表現は使用しない。
4. 広告主素材を掲載する場合は、掲載可否、審査責任、掲載期間、料金、QR計測範囲を別途確認する。

### 第6条 データ利用範囲

1. 本テスト導入で取得または記録するデータは、原則として以下に限定する。

| データ | 利用目的 |
| --- | --- |
| 放映ログ | 表示回数、表示安定性、月次報告の確認 |
| QRアクセスログ | QR反応、時間帯別反応、改善提案の確認 |
| 端末ログ | 障害調査、表示停止原因、運用改善 |
| 設置写真/表示動画 | 設置確認、社内報告、事例化可否判断 |
| 担当者フィードバック | 継続利用判断、改善提案 |

2. 本テスト導入では、来館者を個人として識別するデータ、顔画像、録画データ、個人の行動履歴を取得しないことを標準とする。
3. 個人情報を取得する必要が生じる場合は、取得前に利用目的、取得項目、保存期間、第三者提供の有無、管理方法を別途書面で確認する。
4. 乙は、本テスト導入で取得したデータを、表示改善、障害対応、月次レポート、提案改善、事例作成のために利用できる。
5. 甲の社名、施設名、写真、動画、数値、コメントを外部公開する場合は、第7条の事例掲載条件に従う。

### 第7条 事例掲載・写真動画利用

1. 事例掲載、設置写真、表示動画、担当者コメント、数値結果の外部公開可否は、以下の記入欄で定める。
2. 外部公開の範囲は、営業資料、Webサイト、SNS、プレスリリース、提案資料、広告主向け媒体資料などに分けて確認する。
3. 公開前に甲の確認が必要な場合、乙は公開予定内容を甲へ提示し、承認を得る。

**記入欄**

| 項目 | 可否 |
| --- | --- |
| 施設名の掲載 | 可 / 不可 / 事前確認 |
| 設置写真の掲載 | 可 / 不可 / 事前確認 |
| 表示動画の掲載 | 可 / 不可 / 事前確認 |
| 担当者コメント掲載 | 可 / 不可 / 事前確認 |
| 数値結果の掲載 | 可 / 不可 / 匿名なら可 / 事前確認 |
| 営業資料での利用 | 可 / 不可 / 事前確認 |
| Web/SNSでの公開 | 可 / 不可 / 事前確認 |

### 第8条 保守・障害対応

1. テスト導入期間中、乙は合理的な範囲で遠隔確認、再起動支援、表示内容の軽微な修正を行う。
2. 対応時間は、原則として平日営業時間内とする。ただし、別途合意した場合を除く。
3. 停電、通信断、施設側設備、第三者サービス、天災、事故、標準外作業に起因する障害は、乙の責任範囲外とする。
4. 現地対応が必要な場合は、甲乙協議のうえ対応日、費用、作業範囲を決定する。

### 第9条 破損・紛失

1. 乙が提供した機材の所有権は、別途合意がない限り乙に帰属する。
2. 通常利用に伴う不具合は、乙が確認し、必要に応じて修理または交換を検討する。
3. 甲または第三者の故意、過失、不適切な取扱い、無断移動、無断配線変更により機材が破損または紛失した場合、修理費、交換費、撤去費を甲乙協議のうえ精算する。
4. 機材の移動、取り外し、電源断、ネットワーク変更を行う場合は、事前に乙へ連絡する。

### 第10条 解約・中止・撤去

1. 甲または乙は、相手方に重大な支障がある場合、協議のうえ本テスト導入を中止できる。
2. 中止または終了時、乙は撤去日を甲と調整し、乙が提供した機材を回収する。
3. 撤去時には、端末内の設定、ログ、素材、認証情報を確認し、必要に応じて削除または回収する。
4. 甲の都合により撤去日が延期される場合、保管責任、電源管理、延長費用を別途協議する。
5. 有償PoCの場合、途中解約時の返金有無、日割り精算有無、実費精算条件を記入欄に定める。

**記入欄**

| 項目 | 内容 |
| --- | --- |
| 中途解約 | 可 / 不可 / 協議 |
| 解約予告 | XX日前 |
| 返金 | なし / 日割り / 協議 |
| 撤去費 | 含む / 別途 |
| 撤去期限 | 終了後XX営業日以内 |

### 第11条 本契約への移行

1. テスト導入終了後、乙は必要に応じて結果レポートと本導入提案を提示する。
2. 本契約へ移行する場合、初期費用、月額費用、契約期間、施工範囲、保守範囲、広告運用、データ利用、支払条件を別途合意する。
3. テスト導入の実施は、本契約の締結を保証するものではない。

### 第12条 協議

本合意書に定めのない事項または解釈に疑義が生じた場合、甲乙は誠実に協議し、解決する。

## 法務確認メモ

このテンプレートは、標準では来館者を個人として識別するデータを取得しない前提で作る。

個人情報を取得する設計へ変更する場合は、以下を法務確認する。

- 利用目的を具体的に特定し、本人が合理的に想定できる説明になっているか
- 利用目的の通知または公表が必要か
- 保存期間、第三者提供、委託、削除方法が決まっているか
- QRフォームや問い合わせフォームで取得する氏名、電話番号、メールアドレスの取扱い
- AIカウント、カメラ、録画、行動履歴を使う場合の掲示、同意、取得範囲
