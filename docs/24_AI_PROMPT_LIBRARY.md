# 24. AI生成プロンプト集

## 目的

ミセルの制作・営業・運用・レポート作成をAIで標準化するためのプロンプト集。

AIは属人的な制作を減らし、月額運用の粗利を上げるために使う。

## 共通入力項目

AIに渡す基本情報：

- 施設名
- 業態
- 設置場所
- 表示目的
- 商品/サービス名
- 価格
- 掲載期間
- ターゲット
- QR誘導先
- 写真/動画素材の有無
- トーン
- NG表現

## 1. 3面構成案生成

### Prompt

あなたは3連サイネージのクリエイティブディレクターです。
以下の情報をもとに、左画面・中央画面・右画面の3面構成案を5パターン作ってください。

条件：
- 左画面は導入または補足
- 中央画面はメイン訴求
- 右画面はQR、価格、CTAなど行動導線
- 文字は短く、大きく
- 音声なしでも伝わる
- 15秒表示を想定

入力情報：
- 施設名：
- 業態：
- 商品/サービス：
- 目的：
- ターゲット：
- 価格：
- QR誘導先：

出力形式：
1. パターン名
2. 左画面
3. 中央画面
4. 右画面
5. 狙い

## 2. 15秒動画台本生成

### Prompt

3連サイネージ用の15秒動画台本を作ってください。

条件：
- 0〜3秒：目を引く
- 3〜10秒：メイン訴求
- 10〜15秒：QR/CTA
- 左、中央、右の画面別に指示を書く
- ナレーションなしでも伝わる

入力情報：
- 施設名：
- 商品/サービス：
- 目的：
- 表示場所：
- ターゲット：
- CTA：

出力形式：
- 秒数
- 左画面
- 中央画面
- 右画面
- テロップ
- 動きの指示

## 3. POPタグ文言生成

### Prompt

ミセルPOPタグ用の文言を作ってください。

条件：
- 棚前で一瞬で伝わる
- 商品名、短いコピー、QR誘導文を作る
- 価格がある場合は自然に入れる
- 90mm x 40mm程度の小さいタグを想定

入力情報：
- 商品名：
- 価格：
- 特徴：
- ターゲット：
- QR誘導先：

出力形式：
- 見出し
- 一言コピー
- QR誘導文
- 代替案3つ

## 4. 広告LP生成

### Prompt

ミセルのQR遷移先として使う簡易LPの構成と文章を作ってください。

条件：
- スマホ閲覧前提
- ファーストビューで価値が伝わる
- CTAを明確にする
- 誇大表現を避ける

入力情報：
- 広告主名：
- 商品/サービス：
- 価格：
- 特典：
- 掲載施設：
- CTA：

出力形式：
1. ファーストビュー見出し
2. サブコピー
3. 特徴3つ
4. 利用方法
5. CTA文
6. 注意書き

## 5. 月次レポート文章生成

### Prompt

以下の放映ログとQR計測結果をもとに、広告主向けの月次レポート文章を作ってください。

条件：
- 事実と仮説を分ける
- 数値を明記する
- 断定しすぎない
- 次月の改善案を3つ出す

入力情報：
- 掲載期間：
- 放映回数：
- QR読み取り数：
- 最も反応が多かった時間帯：
- 表示パターン：
- 広告内容：

出力形式：
1. 今月の結果
2. 反応傾向
3. 仮説
4. 次月改善案
5. 継続提案

## 6. 営業メール生成

### Prompt

以下の施設に対して、連結サイネージ「ミセル」のテスト導入を提案する営業メールを作ってください。

条件：
- 売り込みすぎない
- 施設ごとの課題に寄せる
- テスト導入の相談につなげる
- 300〜500文字

入力情報：
- 施設名：
- 業態：
- 想定課題：
- 提案価値：
- 希望アクション：

出力形式：
- 件名
- 本文

## 7. 買い手向けピッチ生成

### Prompt

以下の買い手候補に対して、ミセルを買収/資本提携する価値を説明するピッチ文を作ってください。

条件：
- 買い手の既存事業とのシナジーを明確にする
- ミセルの資産を強調する
- 10億円評価につながる論点を入れる

入力情報：
- 買い手候補：
- 業種：
- 既存顧客：
- ミセルの導入店舗数：
- MRR：
- 広告掲載率：

出力形式：
1. 一言価値
2. シナジー
3. 買収後の成長仮説
4. 主要KPI
5. 次アクション

## 運用ルール

- AI出力をそのまま使わず、人間が必ず確認する
- 価格、効能、法律に関わる表現は特に確認する
- 数値はログや資料に基づく
- 生成物はテンプレートとして蓄積する
