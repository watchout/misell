# 32. 代理店・施工パートナー戦略

## 目的

ミセルは、代表者がすべて施工・営業・運用を行うとスケールしない。

10億円バイアウトを目指すには、早期に代理店、施工パートナー、広告販売パートナーを設計する。

## パートナー種別

### 1. 販売代理店

対象：

- サイネージ販売会社
- 弱電工事会社
- オフィス/店舗設備会社
- ホテル/観光向け営業会社
- 広告代理店

役割：

- 見込み客紹介
- 初回商談
- デモ同席
- 契約支援

報酬設計：

- 実条件はpublic repositoryに置かない
- private storeで案件別、期間別、役割別に承認管理する
- 顧客提示価格は `docs/data/pricing/misell_pricing.public.yaml` を参照する

### 2. 施工パートナー

対象：

- 弱電工事会社
- 電気工事会社
- AV施工会社
- サイネージ設置会社

役割：

- 現調
- 取付
- 配線
- 撤去
- 保守訪問

管理方法：

- 標準施工仕様書
- チェックリスト
- 写真報告
- 検収基準
- 標準単価表

### 3. 広告販売パートナー

対象：

- 地域広告会社
- フリーペーパー
- 商工会関係
- イベント会社
- 店舗支援会社

役割：

- 地域広告主獲得
- 掲載提案
- 素材回収
- 継続提案

報酬設計：

- 実条件はpublic repositoryに置かない
- private storeで広告主対応範囲、掲載審査範囲、継続対応範囲を分けて承認管理する

### 4. 制作パートナー

対象：

- 動画制作者
- デザイナー
- Canva/AI制作オペレーター
- SNS運用者

役割：

- 3面動画制作
- POPタグ制作
- LP制作
- 月次素材更新

報酬設計：

- テンプレート制作単価
- 月額運用単価
- 案件別制作費

## パートナー化の条件

必須：

- 標準仕様を守れる
- 報告ができる
- 顧客対応品質を保てる
- 価格を勝手に崩さない
- 事例化に協力できる

## 代理店に渡す資料

- 1枚資料
- 提案書
- デモ動画
- 価格表
- 施工範囲表
- FAQ
- 契約フロー
- パートナー条件表はpublic repository外で管理する

## 施工パートナーに渡す資料

- 現調チェックリスト
- 標準施工仕様
- 写真報告フォーマット
- 配線図
- 端末設置手順
- トラブル対応表

## 広告販売パートナーに渡す資料

- 媒体資料
- 広告枠価格は承認済みのpricing dataを参照する
- 掲載事例
- QRレポート例
- 広告審査基準

## パートナー展開の順番

### Phase 1

まずは自社主導で1〜3件導入し、標準を作る。

### Phase 2

施工だけ外注化する。

### Phase 3

紹介代理店を1〜3社作る。

### Phase 4

広告販売パートナーを作る。

### Phase 5

地域ごとの施工/販売網を整備する。

## KPI

- パートナー数
- パートナー経由商談数
- パートナー経由受注数
- 施工品質
- クレーム件数
- 広告販売額
- パートナー継続率

## 注意

早期に代理店を増やしすぎない。

商品仕様、価格、施工品質、レポート品質が固まる前に広げるとブランド毀損につながる。

まずは直販で型を作り、再現性が確認できてから代理店展開する。
