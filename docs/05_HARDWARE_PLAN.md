# 05. Hardware Plan

## MVP端末

購入済み：

- NiPoGi Ryzen 4300U
- RAM 16GB
- SSD 256GB
- OSはUbuntuへ移行予定

MVP用途では十分。目的は本番品質の検証ではなく、3連表示、素材入れ替え、タイマー配信、営業デモを成立させること。

## MVPで確認すること

- HDMI / DP / USB-Cの3画面出力
- Ubuntuでの3画面認識
- 6時間以上の連続再生
- 動画3本同時再生
- 3面wide動画再生
- 熱、ファン音、フリーズ有無
- LAN素材転送速度
- 再起動後の復旧

## MVP周辺機器

最低限：

- HDMIケーブル
- DisplayPort to HDMIケーブル
- USB-C to HDMIケーブル
- 有線LANケーブル
- Ubuntuインストール用USBメモリ

推奨：

- スマートプラグ
- HDMI EDIDエミュレータ
- 小型UPS
- モニター3枚
- VESA固定具

## 本番標準機候補

### ASUS NUC 14 Pro系

標準本番機候補。

選定理由：

- Ubuntu対応を前提にしやすい
- HDMIとThunderbolt/USB-C出力が豊富
- 2.5G LANを持つモデルがある
- 法人向けに説明しやすい
- 長期運用の印象が良い

想定構成：

- RAM 32GB
- NVMe SSD 1TB
- Ubuntu 24.04 LTS
- HDMI x 2 + USB-C/Thunderbolt出力

価格帯：本体、RAM、SSD込みでおおむね10万〜16万円を想定。実際の価格は購入時点の見積で確認する。

### MINISFORUM MS-01系

AI拡張機候補。

選定理由：

- 拡張性が高い
- メモリ、SSD、ネットワークに余力がある
- サイネージ端末兼エッジサーバーにできる
- 将来のAI処理、ローカルRAG、カメラ解析に向く

ただし、標準サイネージ用途ではオーバースペックになりやすい。

## 商用専用機候補

### BrightSign XC4055

4 HDMI出力を持つ業務用マルチスクリーンプレイヤー。

高信頼プランや法人向け提案では候補になる。

メリット：

- サイネージ専用機として説明しやすい
- 4画面出力
- 無人運用に強い
- Windows Update事故がない

デメリット：

- 本体価格が高い
- Linux自社プレイヤーとは別系統になる
- 自由なWeb/AI拡張では制約が出る可能性

## 産業用PC候補

Advantechなどの4出力サイネージプレイヤーは、高信頼・大量導入時の候補。

ただし、価格は要見積となるものが多く、MVPでは採用しない。

## 本番構成の3ランク

### Lite

- NiPoGi/同等ミニPC
- 小規模実証・低予算案件
- 初期費用を抑える

### Standard

- ASUS NUC 14 Pro相当
- Ubuntu標準端末
- 3連サイネージ運用の主力

### AI Edge

- MINISFORUM MS-01相当
- AIカメラ、RAG、センサー連携、ローカル分析向け

## 選定ルール

1. 3画面出力がGPU直結であること
2. USB-Cは映像出力対応であること
3. 有線LANがあること
4. BIOSで停電復帰後の自動起動が設定できること
5. Ubuntuで検証できること
6. 端末を隠せるサイズであること
7. 熱暴走しにくいこと
8. 同一構成を複数台調達できること

## 買ってはいけない構成

- HDMI分配器だけ
- Androidスティック3本
- USB-A to HDMIで3画面を無理に出す構成
- ゲートウェイやAPIがない電子棚札タグ単体
- 3画面出力の記載が曖昧なミニPC

## 将来のAI端末化

将来的にミセル端末は、単なるサイネージプレイヤーではなく、店舗内AIエッジ端末になる可能性がある。

想定機能：

- AIコンテンツ生成
- QR反応分析
- 月次レポート生成
- 匿名人数カウント
- 滞留推定
- 混雑検知
- 多言語AI案内
- 店舗内FAQ/RAG

そのため、本番標準機は処理性能に余裕を持たせる。
