### POC価格の条件

POC価格は値引きではない。初期導入パートナーとして、以下に協力してもらう対価である。

- 設置写真/動画の撮影許可
- 導入事例化
- 顧客コメント取得
- 月1回程度のフィードバック
- 匿名化した稼働データ・QR反応データの活用
- 業種別テンプレート開発への協力
- 新機能テスト協力
- 見込み顧客への見学/紹介協力。ただし事前相談必須

### 30日立ち上げ保証

保証するのは売上、集客、広告収益、QRクリック数ではない。保証するのは運用立ち上げである。

```text
必要素材・QR誘導先URL・接続環境が揃った日を起点として、30日以内に初期配信環境を立ち上げます。

当社都合により、30日以内に「自社更新・QR計測・3連表示」が可能な状態を提供できなかった場合、翌月の月額利用料を無料とします。
```

30日以内に作る状態:

| 項目 | 達成内容 |
| --- | --- |
| 3連表示 | 案内・訴求・QRを同時表示できる |
| Misell Studio | 顧客がログインできる |
| 素材更新 | 顧客が自分で素材を差し替えられる |
| スケジュール | 曜日/時間帯配信ができる |
| QR計測 | QRを表示し、クリック数を確認できる |
| 端末監視 | オンライン/オフラインを確認できる |
| 初回テンプレート | 業種別の初期表示パターンが入る |
| 操作説明 | 顧客担当者が基本操作を理解する |

### 含まれるもの

初期導入費に含む: 導入ヒアリング、Misell Studioアカウント発行、POC用配信端末の初期設定、3連表示初期設定、初期テンプレート設定、初回素材登録サポート、QRコード初期設定、テスト配信、操作説明。

月額利用料に含む: Misell Studio利用、POC用配信端末レンタル、顧客セルフ更新機能、スケジュール配信、3連表示、簡易死活監視、簡易再生ログ、QRクリック確認、ソフトウェア更新、メール/チャットサポート。

### 含まれないもの

モニター本体、壁掛け金具/天吊り金具、電源工事、LAN配線工事、下地補強、現地施工、現地訪問サポート、コンテンツ制作、動画編集、素材更新代行、詳細月次レポート作成、広告主向けレポート、現地駆けつけ保守、AIカメラ/AI分析。必要な場合は別途見積もる。

### 禁止表現

以下は資料・LP・営業トークで使わない。

- 売上が上がります
- 集客が増えます
- QR読み取り数が増えます
- 広告収益が出ます
- 必ず広告主がつきます
- AIが売れる販促を自動で作ります
- 視認者数を保証します
- インプレッションを保証します

推奨表現:

```text
QR導線の反応を確認できます。
放映ログとQR反応を見ながら、次回の表示改善につなげます。
広告枠化は将来拡張として検証できます。
```

## 1. 募集LP原稿

### 目的

POC Partnerの問い合わせを獲得する。LP上で決済まではさせない。CTAは「無料診断を申し込む」「設置候補場所を相談する」に統一する。

### セクション構成と原稿

#### Hero

見出し:

```text
施設の“見られる場所”を、30日でインパクトある3連訴求面へ。
```

サブコピー:

```text
紙ポスターや1画面モニターでは埋もれる案内・キャンペーン・QR導線を、3連サイネージで「見せる・伝える・行動につなげる」面に変えます。
```

価格表示:

```text
初期導入パートナー限定
通常 初期398,000円 / 月額39,800円
→ 初期150,000円 / 月額19,800円
```

CTA:

```text
無料で設置場所を診断する
```

補足:

```text
価格は税別。モニター・施工・電源/LAN工事・制作代行は別途見積です。
```

Heroビジュアル: 16:9のテレビモニターを3台横並び。左画面は館内案内、中央画面はキャンペーン訴求、右画面はQR誘導。背景はホテルロビーまたは受付。価格や比較表を画像内へ入れない。

#### Problem

見出し:

```text
1画面では、情報が“順番待ち”になります。
```

本文:

```text
一般的な1画面サイネージでは、案内、キャンペーン、QRが時間で切り替わります。来館者がQRを見る前に通り過ぎたり、重要な案内が表示される前に見落とされたりします。
```

対比表:

| 一般的な1画面 | Misell 3連 |
| --- | --- |
| 案内→キャンペーン→QRが順番に切り替わる | 左:案内、中央:訴求、右:QRを同時表示 |
| QRが出る前に通り過ぎる | QRを常時表示できる |
| 動画とQRが画面を奪い合う | 中央で見せて右で誘導できる |
| 効果が見えにくい | QR反応を確認できる |

締めコピー:

```text
1画面は、順番に見せる。
3連は、同時に見せて、すぐ行動につなげる。
```

#### Solution

見出し:

```text
Misellは、施設内の見られている場所を3連メディアに変えます。
```

本文:

```text
Misellは、3連サイネージを設置するだけのサービスではありません。施設内の受付、ロビー、待合、通路、売店前など、来館者が自然に見る場所を、案内・販促・予約・LINE登録・広告枠へつながる訴求面に変えるサービスです。
```

3つの価値: 目立つ、伝わる、測れる。

#### Features

カード表示: 3連表示、3面ワイド表示、顧客セルフ更新、スケジュール配信、QR計測、端末監視、簡易再生ログ、業種別テンプレート、将来の広告枠化、将来のAI販促提案。

注記:

```text
広告枠化、詳細レポート、AI分析はPOC価格内の標準提供ではなく、別途オプションまたは将来拡張です。
```

#### Industry Slots

限定枠表を掲載し、業種別の刺さる理由を以下の表で示す。

| 業種 | 刺さる理由 |
| --- | --- |
| ホテル・宿泊施設 | 館内案内、朝食/レストラン、周辺観光、予約/LINE/アンケートQRを同時に出せる |
| カラオケ・エンタメ | 1画面では案内、3連なら推し活・誕生日・コラボを演出にできる |
| ジム・温浴・サウナ | 館内案内、回数券、飲食/物販、LINE/予約QRを同時に出せる |
| 貸会議室・イベント会場 | 本日の案内、スポンサー表示、次回予約QRを同時に出せる |

| クリニック・待合 | 診療案内、自由診療、予約/LINE登録QRを同時に出せる |

#### Offer

価格表、POC価格の条件、含まれるもの/含まれないもの、30日保証を掲載する。

#### Flow

| Step | 内容 | 目安 |
| ---: | --- | --- |
| 1 | 無料診断 | 15〜30分 |
| 2 | 設置場所・表示内容の確認 | 1〜3営業日 |
| 3 | POC条件確認・申込 | 1営業日 |

## 4. 業種別モックアップ制作指示

共通仕様: 3台の16:9テレビモニターを横並び。同じサイズ。現実の施設に設置されているように見せる。インフォグラフィックにしない。価格や説明文字を画像内に入れない。解像度は横長16:9、推奨1920x1080。人物が写る場合は個人が特定できない構図にする。

### ホテル・宿泊施設

背景: 清潔感のあるホテルロビー。受付カウンター近くの壁面に3台の横型モニター。左は館内案内、中央は朝食/周辺観光、右はLINE/周辺マップ/アンケートQR。

### カラオケ・エンタメ

背景: カラオケ店舗の受付前。左は本日のイベント/コラボ案内、中央は推し活・誕生日プラン、右は予約QR/フード注文QR/特典QR。

### ジム・温浴・サウナ

背景: 温浴施設またはジム受付横。左は館内案内/混雑時間、中央はサウナ・回数券・キャンペーン、右は予約QR/LINE登録/クーポン。

### 貸会議室・イベント会場

背景: 会議室フロアの受付またはエレベーターホール。左は本日のイベント一覧、中央は開催中イベント/スポンサー、右は会場案内QR/次回予約QR。

### 飲食・小売

背景: 飲食店または小売店の入口付近。左はメニュー/ランキング、中央は季節限定商品、右は注文QR/クーポンQR/LINE登録。

### クリニック・待合

背景: クリニックまたは歯科の待合。落ち着いた色味。左は診療案内、中央は自由診療/予防ケアの案内、右は予約QR/LINE登録/問診票QR。医療広告表現は慎重にし、効果効能の断定、ビフォーアフター、過度な煽りを使わない。

## 12. FAQ・反論処理集

### Q1. モニターは含まれますか？

A: POC価格にはモニター本体は含まれません。既存モニターを使える場合はそのまま活用できます。新規モニター、壁掛け金具、施工が必要な場合は別途見積となります。

### Q2. 工事は含まれますか？

A: 電源工事、LAN配線工事、下地補強、壁掛け/天吊り施工、夜間作業は含まれません。設置条件を確認したうえで、必要な場合のみ別途見積します。

### Q3. 自社で更新できますか？

A: はい。POC版では、顧客側で素材を差し替えられる状態を作ります。初回は操作説明も行います。ただし、継続的な制作代行や大量の素材更新は別途オプションです。

### Q4. QRの効果は保証されますか？

A: 保証しません。Misellで提供するのは、QRを表示し、反応を確認できる状態です。QR読み取り数や予約数、売上増加は保証対象外です。

### Q5. 広告収益は保証されますか？

A: 保証しません。POCでは広告枠化の可能性を検証できますが、広告主獲得や広告収益は保証しません。広告枠運用を行う場合は、Mediaプランまたは別途支援として設計します。

### Q6. なぜ安いのですか？

A: 単なる値引きではありません。初期導入パートナーとして、設置写真/動画、導入事例、月1回のフィードバック、匿名化した稼働データ・QR反応データの活用にご協力いただくことを前提にした限定価格です。

### Q7. 途中解約できますか？

A: POC価格は端末レンタルと初期設定を含むため、原則として最低利用期間を設定します。標準では24か月を推奨しますが、案件条件により事前に協議します。

### Q8. AI分析も使えますか？

A: AI分析やAIカメラはPOC価格には含まれません。まずは3連表示、自社更新、QR計測、端末監視を立ち上げ、必要に応じて上位オプションとして検討します。
