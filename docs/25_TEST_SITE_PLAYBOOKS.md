# 25. テスト導入先別プレイブック

## 目的

安心お宿、バリアン、パセラ、ビジョンセンターへのテスト導入を、個別提案として組み立てる。

各施設で「何を見せるか」「どこに置くか」「何をKPIにするか」を事前に定義する。

## 共通テスト導入条件

- 期間：2〜4週間
- 目的：視認性、運用価値、QR反応、月額化可能性の検証
- 成果物：設置写真、表示動画、QRレポート、担当者コメント
- 条件：事例化の相談、フィードバック協力

## 安心お宿プレイブック

### 仮説

館内案内、サウナ/温浴回遊、近隣広告、多言語案内に価値がある。

### 設置候補

- フロント前
- エレベーターホール
- 休憩スペース
- 朝食/ドリンクエリア

### 初期表示内容

- 館内案内
- 大浴場/サウナ案内
- 朝食/ドリンク案内
- 周辺飲食広告
- QR館内マップ

### KPI

- 館内案内QR
- 周辺飲食QR
- フロント問い合わせ削減感
- スタッフ評価
- 広告主候補数

### 有償化提案

館内案内 + 近隣広告枠 + 多言語案内を月額プラン化する。

## バリアンプレイブック

### 仮説

記念日、ルームサービス、アメニティ、予約導線に価値がある。

### 設置候補

- フロント
- 待合スペース
- エレベーター前

### 初期表示内容

- 記念日プラン
- ルームサービスランキング
- アメニティ案内
- 予約/リピート導線
- 近隣飲食/レジャー広告

### KPI

- ルームサービスQR
- 記念日プランQR
- オプション利用数
- 待合での視認性
- 客単価アップ仮説

### 有償化提案

客単価アップ支援 + リピート予約導線 + 近隣広告枠として提案する。

## パセラプレイブック

### 仮説

フード/ドリンク、推し活、コラボ、系列施設回遊に価値がある。

### 設置候補

- 受付
- エレベーターホール
- 店舗入口
- イベントスペース

### 初期表示内容

- フード/ドリンクおすすめ
- コラボ/推し活告知
- 系列施設案内
- イベント案内
- 予約QR

### KPI

- フード/ドリンクQR
- コラボ予約QR
- 系列施設送客
- スポンサー候補数

### 有償化提案

店内販促 + コラボ告知 + スポンサー広告枠として提案する。

## ビジョンセンタープレイブック

### 仮説

会場案内、当日イベント表示、スポンサー広告、配信パック訴求に価値がある。

### 設置候補

- 受付
- エレベーターホール
- 会議室前
- 共用スペース

### 初期表示内容

- 本日のイベント一覧
- 会場案内
- 配信パック広告
- スポンサー枠
- QR問い合わせ

### KPI

- 配信パックQR
- 会場案内QR
- 受付問い合わせ削減感
- スポンサー広告候補数
- イベント主催者評価

### 有償化提案

会場メディア + 配信パック送客 + スポンサー広告枠として提案する。

## テスト導入後の共通レポート

- 設置場所
- 表示内容
- 放映回数
- QR読み取り数
- 担当者コメント
- 問題点
- 改善案
- 有償化提案

## 次アクション

1. 各施設向けの60秒デモ動画を作る
2. 施設ごとの1枚提案書を作る
3. 設置場所の写真を入手する
4. テスト導入条件を提示する
5. 1件目の導入を最優先で決める
