# 77. Misell PoC Partner Pricing and Buyout Strategy

## 目的

このドキュメントは、Misellの初期導入パートナー制度、PoC価格、端末レンタル資産スキーム、代理店手数料、業種別募集枠、70施設シミュレーション、バイアウトに向けた成長戦略をまとめる。

本資料は、提案書・営業資料・価格表を作成する前の社内正本として扱う。

## 結論

Misellは、初期フェーズでは複数プランを羅列せず、まず **Misell 初期導入パートナープログラム** を主力オファーとして販売する。

PoC価格でも利益が出る構造にし、安売りではなく、事例化・テンプレート化・運用データ取得・MRR形成を同時に進める。

## 商品コンセプト

Misellは、3連サイネージを設置する会社ではなく、施設内の「見られている場所」を、更新できる・測れる販促メディアに変えるサービスである。

提供価値:

- 3連表示
- 顧客セルフ更新
- Misell Studio
- QR計測
- 端末監視
- 簡易ログ
- 業種別テンプレート
- 将来の広告枠化
- 将来のAI分析

## 初期導入パートナープログラム

### 外向け名称

**Misell 初期導入パートナープログラム**

内部名:

**PoC Partner**

### 価格

| 項目 | 通常価格 | PoC Partner価格 |
| --- | ---: | ---: |
| 初期導入費 | 398,000円 | 150,000円 |
| 月額利用料 | 39,800円 | 19,800円 |

価格は税別。

### 前提

- 端末はMisell所有のレンタル扱い
- PoC端末は4〜5万円級の検証用端末を想定
- 顧客セルフ更新前提
- モニター、金具、施工、現地訪問は別見積
- 更新代行、制作、詳細レポート、広告主レポート、AI分析は別オプション

### PoC価格の意味

PoC価格は値引きではなく、初期導入パートナーとして以下に協力してもらう対価である。

- 設置写真/動画の撮影許可
- 導入事例化
- 顧客コメント取得
- 月1回程度のフィードバック
- 匿名化した稼働データ・QR反応データの活用
- 見込み顧客への見学/紹介協力、要事前相談
- 新機能テスト協力

### 契約期間

最低契約期間は **24か月** を推奨する。

理由:

- 端末レンタル原価の回収
- 端末資産の管理
- サポート工数の平準化
- 顧客との関係構築
- 事例化・テンプレート化の時間確保

### 24か月後の扱い

PoC Partner価格は、初期導入パートナー条件を満たす限り継続適用できる。

ただし、24か月経過後に以下を確認し、継続条件を見直せるようにする。

| 状況 | 対応 |
| --- | --- |
| 条件を満たす | PoC価格継続可 |
| 事例化・撮影協力が不可になった | Standard価格へ移行検討 |
| サポート負荷が高い | Managed Option追加 |
| 端末更新が必要 | Standard端末へ切り替え |
| 広告枠化を行う | Media Option追加 |
| 業態テンプレート化に寄与しない | 継続条件を再協議 |

## 初期費用150,000円の原価根拠

PoC端末は月額レンタルに含めるが、初期導入時には以下の工数が発生する。

| 項目 | 想定原価/工数感 |
| --- | ---: |
| PoC端末取得 | 40,000〜50,000円 |
| 端末キッティング | 20,000〜30,000円 |
| 3連表示初期設定 | 20,000〜30,000円 |
| Misell Studio設定 | 10,000〜20,000円 |
| 初期テンプレート/QR設定 | 10,000〜20,000円 |
| 操作説明 | 10,000円前後 |
| 発送/事務/予備費 | 5,000〜10,000円 |

想定合計: 115,000〜170,000円

PoC初期費用150,000円は、端末取得費と初期設定工数を概ね回収できる水準として設定する。

## 月額19,800円の原価根拠

月額19,800円は、顧客セルフ更新・現地対応なし・メール/チャットサポート前提で成立する。

月額に含めるもの:

- Misell Studio利用
- PoC用端末レンタル
- 顧客セルフ更新
- スケジュール配信
- 3連表示
- 簡易死活監視
- 簡易再生ログ
- QRクリック確認
- ソフトウェア更新
- メール/チャットサポート

月額に含めないもの:

- 現地訪問
- 現地駆けつけ
- 素材更新代行
- コンテンツ制作
- 詳細月次レポート
- 広告主レポート
- 電源/LAN工事
- モニター/金具/施工
- AI分析

## 端末レンタル資産スキーム

### 基本方針

PoC端末は販売しない。Misell所有のレンタル端末として扱う。

狙い:

- 端末所有権をMisell側に残す
- 解約時に回収できる
- 別施設へ再投入できる
- Standard端末へのアップグレード提案ができる
- 端末を資産として管理できる
- 端末品質をMisell側で統制できる

### 端末階層

| 区分 | 端末 | 用途 |
| --- | --- | --- |
| PoC端末 | 4〜5万円級 | 表示・Studio・QR・運用検証 |
| Standard端末 | 16〜18万円級 | 商用24時間運用 |
| Premium端末 | BrightSign / 産業用PC | 高信頼/大手/止められない現場 |

### 会計上の注意

端末をMisell所有資産とする場合、減価償却・少額資産処理・一括償却資産などの扱いは税理士確認が必要。

PoC端末が1台40,000〜50,000円の場合、70台で280万〜350万円規模の端末取得が発生する。

## 代理店・パートナー手数料

### 基本方針

代理店が一次対応を担う場合、継続手数料を認める。

| パートナー区分 | 初期手数料 | 月額手数料 | 期間 | 条件 |
| --- | ---: | ---: | --- | --- |
| Referral Partner | 20% | なし/5% | 初年度まで | 紹介のみ |
| Sales Partner | 20% | 10% | 12か月 | 商談支援 |
| Operation Partner | 20% | 15% | 継続 | 一次対応あり |
| Reseller Partner | ネット価格 | 自由 | 継続 | 契約/請求/一次対応あり |

### Operation Partnerの一次対応範囲

- 顧客問い合わせ一次受け
- 操作方法の説明
- 素材差し替え方法の案内
- 電源/HDMI/LANの確認依頼
- 再起動案内
- 画面状態の確認
- 顧客との日程調整
- 現地訪問が必要な場合の一次判断

### Misellの二次対応範囲

- システム障害
- 端末ソフト不具合
- リモートログ確認
- Player/CMS修正
- アップデート
- 端末交換判断
- 高度な配信不具合
- QR/レポート不具合

## 70施設シミュレーション

### 70施設すべてPoC Partnerの場合

| 項目 | 計算 | 金額 |
| --- | ---: | ---: |
| 初期売上 | 150,000円 × 70 | 10,500,000円 |
| 月額MRR | 19,800円 × 70 | 1,386,000円/月 |
| 年間ARR | 1,386,000円 × 12 | 16,632,000円/年 |

### 代理店ありの場合

| 項目 | 計算 | 金額 |
| --- | ---: | ---: |
| 初期代理店手数料 | 10,500,000円 × 20% | 2,100,000円 |
| 初期残 | 10,500,000円 - 2,100,000円 | 8,400,000円 |
| 月額代理店手数料 | 1,386,000円 × 15% | 207,900円/月 |
| Misell側月額残 | 1,386,000円 - 207,900円 | 1,178,100円/月 |

### 端末原価50,000円の場合

| 項目 | 計算 | 金額 |
| --- | ---: | ---: |
| 端末仕入総額 | 50,000円 × 70 | 3,500,000円 |
| 初期残から端末仕入控除後 | 8,400,000円 - 3,500,000円 | 4,900,000円 |

### 月次原価例

前提:

- 端末原価 50,000円
- 端末償却 24か月
- クラウド/監視原価 2,000円/施設/月
- サポート 0.5時間/施設/月
- サポート時給換算 6,000円

| 月額原価 | 1施設 | 70施設 |
| --- | ---: | ---: |
| 端末償却 | 2,083円 | 145,810円 |
| クラウド/監視 | 2,000円 | 140,000円 |
| サポート | 3,000円 | 210,000円 |
| 合計 | 7,083円 | 495,810円 |

代理店控除後MRRから月次原価を引いた粗利:

```text
1,178,100円 - 495,810円 = 682,290円/月
```

このため、代理店あり・端末償却・クラウド・サポート0.5h/施設/月でも、月68万円程度の粗利が見込める。

## サポート時間KPI

PoC Partnerの採算を守る最大のKPIは、1施設あたりのサポート時間である。

| サポート時間 | 評価 |
| ---: | --- |
| 0.25h/月 | 非常に良い |
| 0.5h/月 | 許容範囲 |
| 1.0h/月 | 注意。粗利を圧迫 |
| 2.0h/月 | 危険。サポート地獄化 |

目標:

```text
1施設あたり月30分以内
```

このため、セルフ運用、業種別テンプレート、操作マニュアル、FAQ、代理店一次対応が重要になる。

## バイアウトに向けた戦略

### 基本思想

Misellのバイアウト価値は、施工売上ではなく、MRR・導入ネットワーク・端末管理基盤・広告/QRデータ・業種別テンプレート・運用ノウハウにある。

目指す状態:

- 継続課金MRR
- 導入施設ネットワーク
- 業種別テンプレート
- 端末監視/遠隔更新基盤
- QR/再生ログ/広告レポート
- 代理店ネットワーク
- Media/AI Edgeへのアップセル余地

### Phase 1: 0〜6か月

目標:

- 初期導入パートナー 10〜20施設
- ホテル/カラオケ/温浴のどれかで強い事例を作る
- PoC端末運用の安定性確認
- サポート時間KPIを測る
- QR計測の実績を作る
- 提案書・事例資料・操作マニュアルを整備

KPI:

- MRR 20万〜40万円
- 稼働率 98%以上
- QR設定率 80%以上
- 事例化 5件以上
- 1施設あたり月サポート30分以内

### Phase 2: 6〜12か月

目標:

- 初期導入パートナー 30〜50施設
- 業種別テンプレートを整備
- 代理店/Operation Partnerの稼働
- Media Optionの実証
- 広告主レポートの簡易版を作る

KPI:

- MRR 60万〜100万円
- ARR 700万〜1,200万円
- 業種別テンプレート 5種以上
- 代理店経由比率 30%以上
- 事例化 15件以上

### Phase 3: 12〜24か月

目標:

- 70施設を達成
- MRR 100万円超
- Media Option/Managed Optionを追加
- 広告枠化を一部施設で実施
- Standard端末/商用端末へのアップセル
- QR/再生ログ/広告レポートを蓄積

KPI:

- MRR 100万〜150万円
- ARR 1,200万〜1,800万円
- 端末稼働率 98%以上
- 解約率 低水準維持
- Media導入 10施設以上
- 広告主 10社以上

### Phase 4: 24〜36か月

目標:

- 150〜300施設
- MRR 300万〜600万円
- Media/AI Edgeのアップセル
- 代理店網拡大
- 買収候補に見せられるKPIを整備

買収候補:

- サイネージ大手
- USEN系/店舗DX企業
- 広告代理店/リテールメディア企業
- ホテル/施設向けSaaS企業
- 弱電/設備系の全国保守会社
- AIカメラ/店舗分析企業
