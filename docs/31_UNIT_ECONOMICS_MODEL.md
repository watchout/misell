# 31. ユニットエコノミクス設計

## 目的

ミセルを10億円バイアウト対象にするには、1店舗あたりの採算が明確である必要がある。

このドキュメントでは、初期費用、月額、広告運用、施工、自社施工時の利益上振れを整理する。

## 収益項目

### 初期収益

- 端末/機材
- ディスプレイ
- 金具
- キッティング
- 初期設定
- 施工
- 初期コンテンツ制作

### 月額収益

- 配信管理
- 監視
- 保守
- 素材更新
- レポート
- 広告運用
- AI制作

### 追加収益

- 動画制作
- POPタグ制作
- 広告主LP制作
- AIカメラ
- 電子棚札連携
- 多店舗本部管理

## 自社施工時の利益上振れ

価格は外注前提で組む。

自社施工できる案件では、外注施工想定原価の一部が利益上振れになる。

例：

- 外注施工想定：200,000円
- 自社施工実費：50,000円
- 上振れ利益：150,000円

ただし、代表者の時間を無償扱いしすぎないこと。将来スケール時は外注前提の採算に戻る。

## 10億評価に必要な経済性

### 目標ユニット

- 平均月額売上：80,000円以上
- 平均月額粗利：40,000円以上
- 粗利率：50%以上
- 解約率：月2%以下
- 契約期間：36か月以上

### 店舗数別MRR

- 10店舗 x 80,000円 = 800,000円
- 30店舗 x 80,000円 = 2,400,000円
- 50店舗 x 100,000円 = 5,000,000円
- 100店舗 x 100,000円 = 10,000,000円

## 改善レバー

- 月額更新回数を増やす
- AIレポートを有料化する
- POPタグを追加する
- 広告主を獲得する
- 多店舗管理費を取る
- 初期制作をテンプレート化して原価を下げる
- 施工を標準化して原価を下げる

## 注意

安く売りすぎると、店舗数が増えても運用負荷だけが増える。

最低でも、1店舗あたり月額粗利4万円以上を目標にする。
