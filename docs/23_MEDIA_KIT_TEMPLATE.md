# 23. 広告主向け媒体資料テンプレート

## 目的

ミセルを広告枠として販売するための媒体資料テンプレート。

店舗向け資料とは分けて、広告主には「どこに、何回、どんな人に、どのような導線で見られるか」を伝える。

## 媒体名

**ミセルMedia**

## 一言説明

ホテル、温浴施設、会議室、観光施設などの館内3連サイネージに広告を掲載できる地域密着型メディア。

## 媒体資料の構成

### 1. 媒体概要

- 媒体名：ミセルMedia
- 掲載場所：施設内3連サイネージ
- 表示形式：3面wide / 中央広告 + QR / 3分割広告
- 掲載単位：1週間 / 1か月
- 計測：放映回数、QR読み取り数、時間帯別反応

### 2. 掲載施設情報

施設ごとに以下を記載。

- 施設名
- 業態
- 所在地
- 月間来館者数
- 主な客層
- 設置場所
- 想定視認シーン
- 営業時間

### 3. 広告枠メニュー

#### ライト枠

- 月額：10,000円〜30,000円
- 15秒素材
- ローテーション掲載
- QR計測
- 簡易レポート

#### スタンダード枠

- 月額：30,000円〜50,000円
- 時間帯指定
- QR/POPタグ連動
- 月次レポート

#### プレミアム枠

- 月額：50,000円〜100,000円
- 3面wide枠
- 優先表示
- 専用クリエイティブ
- 改善提案

### 4. 表示回数の考え方

例：

- 1ロール：5分
- 15秒広告：1時間あたり12回表示
- 12時間営業：1日144回表示
- 30日：月4,320回表示

注意：実際のロール設計により変動するため、媒体資料では「想定表示回数」と明記する。

### 5. QR計測

広告ごとに専用QRを発行する。

計測項目：

- QR読み取り数
- 日別アクセス
- 時間帯別アクセス
- 掲載施設別アクセス

### 6. 掲載可能広告

- 飲食店
- 整体/マッサージ
- 美容室
- 観光体験
- 不動産
- 求人
- スクール
- ジム
- 地域イベント
- 自治体/観光協会

### 7. 掲載審査

掲載前に内容審査を行う。

NG例：

- 虚偽/誇大広告
- 法令違反のおそれ
- 著作権未確認素材
- 施設ブランドを損なう表現
- 公序良俗に反する内容

### 8. 入稿仕様

#### 3面wide

- サイズ：5760 x 1080
- 尺：15秒または30秒
- 形式：mp4推奨

#### 中央広告 + QR

- 中央素材：1920 x 1080
- 右側QR画像またはURL
- 左側補助コピー

#### 静止画

- jpg / png
- 1920 x 1080 または 5760 x 1080

### 9. レポート例

月次レポート項目：

- 掲載期間
- 放映回数
- QR読み取り数
- 時間帯別反応
- 改善提案
- 次月おすすめ表示パターン

### 10. 申込フロー

1. 掲載希望施設/期間を選択
2. 素材入稿
3. 審査
4. QR発行
5. 掲載開始
6. 月次レポート提出

## 営業トーク

「地域の施設内で、実際に滞在しているお客様に広告を届けられます。紙チラシと違い、QRで反応を計測し、月次レポートで改善できます。」

## 注意

初期はインプレッション保証をしない。

最初は、掲載枠 + QR計測 + 月次レポートとして売る。

視認者数保証や属性分析は、AIカウント導入後の上位プランにする。
