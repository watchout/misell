# 00. Project Overview

## 事業名

**連結サイネージ「ミセル -misell-」**

## タグライン

**つないで魅せる。売場をメディアに。**

## 一言説明

ミセルは、3連・多連サイネージを起点に、店舗や施設の空きスペースを、販促・案内・広告収益・AI分析のメディアへ変える運用サービスである。

## 何を売るのか

モニターやPC単体ではなく、以下をセットで提供する。

1. 連結サイネージ端末
2. 3面/多面コンテンツ配信
3. LAN/クラウド素材更新
4. スケジュール配信
5. 運用監視・保守
6. 3面クリエイティブ制作支援
7. 地域広告枠運用
8. QR/POPタグ/電子棚札連動
9. AIによる制作・分析・改善提案

## なぜ今やるのか

大手コンビニや大手小売では、店内サイネージがリテールメディアとして活用され始めている。一方、中小店舗・ホテル・温浴・貸会議室・観光施設では、導入、施工、更新、広告運用までを一気通貫で支援する事業者が不足している。

ミセルは、大手向けの大規模広告ネットワークではなく、中小〜中堅施設向けの「小さく始められる店舗メディア化」を狙う。

## 初期ターゲット

優先順位は以下。

1. 安心お宿
2. バリアン
3. パセラ
4. ビジョンセンター
5. ホテル売店・フロント・朝食会場
6. 温浴施設・サウナ
7. 道の駅・観光施設
8. 飲食ビル・商業施設
9. 地域スーパー・惣菜・テイクアウト

## 競争軸

ミセルはLEDウォール業者や一般サイネージ業者と、画面性能だけで競争しない。

勝負する領域は以下。

- 3面クリエイティブ
- 低初期導入
- 月額運用
- 地域広告枠化
- 導入後の素材更新代行
- QR/POPタグによる効果計測
- AIレポート
- 弱電・施工・運用を含めた一括対応

## MVPの目的

MVPは「完璧なCMS」ではなく、次を検証するための最小構成である。

- 3画面連動表示が安定するか
- 左・中央・右の個別表示ができるか
- LAN経由で素材を入れ替えられるか
- タイマー配信できるか
- 現場導入した時に価値が伝わるか
- テスト導入先で月額運用に発展するか
- 広告枠/POPタグ/QR計測まで拡張できるか

## 2027年6月のゴール

2027年6月時点で、10億円でのバイアウトを狙える状態を作る。

そのために、単なる施工案件ではなく、以下を持つ事業にする。

- 継続課金MRR
- 導入店舗ネットワーク
- 広告主ネットワーク
- コンテンツ運用ノウハウ
- 標準施工仕様
- 端末管理基盤
- 放映ログ/QRログ/広告レポート
- AI制作・AI分析機能
