# 40. 資金調達・補助金・リース活用計画

## 目的

ミセルは初期導入費を下げるほど受注しやすくなる一方、機材・施工・制作の立替負担が発生する。

資金繰りを壊さずに導入件数を増やすため、リース、補助金、融資、協業を活用する。

## 資金が必要になる場面

- デモ機制作
- テスト導入機材
- 量産端末調達
- ディスプレイ/金具調達
- 施工外注費
- 動画制作費
- LP/営業資料制作
- 広告運用人件費
- クラウド化開発

## 基本方針

- 在庫は持ちすぎない
- 受注後調達を基本にする
- 初期を下げる場合は最低契約期間を設定する
- 機材はリース/割賦を活用する
- 月額運用費は自社の継続収益として残す

## リース活用

### 顧客リース

顧客が機材/施工費をリースで支払い、自社は運用月額を別で取る。

メリット：

- 自社の立替が減る
- 顧客の初期負担が下がる
- 契約期間を作りやすい

### 自社リース/レンタル

自社が機材を持ち、顧客へ月額提供する。

メリット：低初期で売りやすい。

デメリット：資金負担が重い。初期は慎重に使う。

## 補助金活用

対象になり得る方向性：

- 店舗DX
- 販促DX
- 多言語対応
- 観光DX
- 省人化
- 業務効率化
- デジタルサイネージ導入
- AI活用

注意：補助金は制度変更があるため、案件ごとに最新要件を確認する。

## 融資活用

融資は、受注見込みとMRRが見えてから使う。

資金使途：

- デモ機複数台
- 初期導入機材
- クラウド化開発
- 営業資料/LP
- 外注制作

## 低初期プランの資金設計

低初期プランでは、必ず以下をセットにする。

- 最低契約期間
- 中途解約時の残債精算
- 機材所有権の明記
- 保守範囲
- 支払遅延時の停止条件

## キャッシュフロー管理

案件ごとに管理する。

- 初期売上
- 初期原価
- 回収月数
- 月額粗利
- 広告粗利
- 施工支払タイミング
- 入金タイミング

## 資金調達のタイミング

### Phase 1

自己資金 + 小額機材でMVP。

### Phase 2

テスト導入1〜3件後に、デモ機/営業資料へ小額投資。

### Phase 3

10店舗が見えたら、運転資金として融資/リース枠を検討。

### Phase 4

30店舗が見えたら、資本業務提携や買収候補との提携を検討。

## 注意

補助金や融資は目的ではない。

ミセルの本質は、継続課金と店舗メディアネットワークの構築である。
