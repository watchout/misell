# 78. Final 30-Day Impact Offer and Proposal Strategy

## 目的

このドキュメントは、Misellの最終的なPoC/初期導入パートナー戦略と、営業提案資料の見せ方をまとめる社内正本である。

既存の `docs/77_POC_PARTNER_PRICING_AND_BUYOUT_STRATEGY.md` は、PoC価格、端末レンタル、70施設シミュレーション、代理店手数料、バイアウト戦略の正本として残す。

本ドキュメントは、外向けオファー、39秒ピッチ、提案資料の最初の3ページ、Claude Design向け指示を固定する。

## 最終結論

Misellの初期PoCは、単なる「3連サイネージのテスト導入」ではなく、以下のオファーとして販売する。

```text
施設の“見られる場所”を、30日でインパクトある3連訴求面へ。
```

売る主語は、機材・CMS・QR計測ではない。

売るべきものは、施設内の一番見られる場所に作る **インパクトある3連訴求面** である。

- 1画面では弱い告知を、3連で強く見せる
- 案内・訴求・QR誘導を同時に表示する
- 紙ポスターや放置モニターでは埋もれる情報を、来館者の目に止まる面に変える
- クラウド更新、QR計測、端末監視は裏側の安心材料として提供する

## 39秒ピッチ

```text
施設内には、来館者が必ず見る場所があります。
でも多くの施設では、そこが紙ポスターや1画面モニターになっていて、案内・キャンペーン・QR導線が埋もれています。

Misellは、その場所に3連のインパクトある訴求面を作り、左で案内、中央で訴求、右でQR誘導を同時に出せるようにします。

30日以内に、クラウド更新・QR計測・端末監視まで使える状態で立ち上げます。

初期導入パートナーは、通常初期398,000円・月額39,800円のところ、初期150,000円・月額19,800円でご案内しています。

まずは、御社のどの場所が3連化に向いているか無料で診断します。
```

## なぜ3連なのか

3連の価値は「画面が多い」ことではない。

1画面では、案内・キャンペーン・QRが順番待ちになる。Misellの3連では、案内・訴求・QR誘導を同時に出せる。

| 一般的な1画面サイネージ | Misell 3連サイネージ |
| --- | --- |
| 案内→キャンペーン→QRが順番に切り替わる | 左:案内、中央:訴求、右:QRを同時表示 |
| QRが出る前に通り過ぎる | QRを常時表示できる |
| 動画とQRが画面を奪い合う | 中央で見せて右で誘導できる |
| 告知が弱く、流して終わりになりやすい | インパクトある訴求面として目を引く |
| 効果が見えにくい | QR反応を確認できる |

一言で言うと:

```text
1画面は、順番に見せる。
3連は、同時に見せて、すぐ行動につなげる。
```

## 商品コンセプト

Misellは、3連サイネージを設置する会社ではない。

施設内の「見られているのに活かせていない場所」を、インパクトある3連訴求面に変え、案内・販促・予約・LINE登録・QR誘導へつなげるサービスである。

提供価値の優先順位:

1. 目立つ
2. 伝わる
3. 行動につながる
4. 測れる
5. 改善できる

機能は以下:

- 3連表示
- 3面ワイド表示
- 顧客セルフ更新
- Misell Studio
- QR計測
- 端末監視
- 簡易ログ
- 業種別テンプレート
- 将来の広告枠化
- 将来のAI分析

| 項目 | 通常価格 | 初期導入パートナー価格 |
| --- | ---: | ---: |
| 初期導入費 | 398,000円 | 150,000円 |
| 月額利用料 | 39,800円 | 19,800円 |

## 価格の位置づけ

この価格は単なる値引きではない。

初期導入パートナー価格は、以下に協力してもらう対価として提供する。

- 設置写真/動画の撮影許可
- 導入事例化
- 顧客コメント取得
- 月1回程度のフィードバック
- 匿名化した稼働データ・QR反応データの活用
- 業種別テンプレート開発への協力
- 新機能テスト協力

## 30日立ち上げ保証

ホルモジ型のオファーとして、リスク逆転を入れる。

ただし、保証するのは売上・集客・QRクリック数ではない。

保証するのは、運用立ち上げである。

### 保証文

```text
必要素材・QR誘導先URL・接続環境が揃った日を起点として、30日以内に初期配信環境を立ち上げます。

当社都合により、30日以内に「自社更新・QR計測・3連表示」が可能な状態を提供できなかった場合、翌月の月額利用料を無料とします。
```

### 30日以内に作る状態

| 項目 | 達成内容 |
| --- | --- |
| 3連表示 | 案内・訴求・QRを同時表示できる |
| Misell Studio | 顧客がログインできる |
| 素材更新 | 顧客が自分で素材を差し替えられる |
| スケジュール | 曜日/時間帯配信ができる |
| QR計測 | QRを表示し、クリック数を確認できる |
| 端末監視 | オンライン/オフラインを確認できる |
| 初回テンプレート | 業種別の初期表示パターンが入る |
| 操作説明 | 顧客担当者が基本操作を理解する |

### 30日保証に含めないもの

- 売上増加
- 広告収益化
- 集客増加
- QRクリック数の保証
- 大量コンテンツ制作
- 現地施工完了
- モニター設置
- 電源/LAN工事
- 高度な広告レポート
- AI分析

### 30日起点の前提

- 既存モニターまたは顧客側で設置済み
- モニター/施工が必要な場合、施工完了後から30日
- 顧客から必要素材が提供されること
- QR誘導先URLが決まっていること
- 担当者が操作説明を受けられること
- PoC端末在庫があること

## 業種別の刺さる理由

| 業種 | 3連が刺さる理由 |
| --- | --- |
| ホテル・宿泊施設 | 館内案内、朝食/レストラン、周辺観光、予約/LINE/アンケートQRを同時に出せる |
| カラオケ・エンタメ | 1画面では案内、3連なら推し活・誕生日・コラボを演出にできる |
| ジム・温浴・サウナ | 館内案内、回数券、飲食/物販、LINE/予約QRを同時に出せる |
| 貸会議室・イベント会場 | 本日の案内、スポンサー表示、次回予約QRを同時に出せる |
| 飲食・小売 | メニュー、キャンペーン、注文/クーポンQRを同時に出せる |
| クリニック・待合 | 診療案内、自由診療、予約/LINE登録QRを同時に出せる |

## 向いている施設

- 来館者が必ず見る壁面・受付・ロビー・待合・通路がある
- 紙ポスターや1画面モニターでは告知が弱いと感じている
- キャンペーンや案内を頻繁に変えたい
- 予約、LINE登録、メニュー、クーポンなどに誘導したい
- 将来的に広告枠化やスポンサー表示を検討したい
- 初期事例化やフィードバックに協力できる

## 向いていない施設

- 表示内容を一切更新する予定がない
- QRや予約導線を使うつもりがない
- ただ動画を流せればよい
- 事例化やフィードバックに協力できない
- 30日立ち上げに必要な素材や環境を用意できない

## 提案資料の考え方

相手は資料をほとんど読まない。

最初に見るのは、おおむね以下の3つである。

1. 表紙の一言
2. 設置イメージ
3. 価格と限定条件

したがって、提案資料は最初の3ページで「話を聞く理由」を作る。

細かい機能説明、FAQ、条件、含まれないものは後ろに回す。

## 提案資料の最初の3ページ

### 1ページ目: 強い約束

```text
Misell 初期導入パートナー募集

施設の“見られる場所”を、インパクトある3連訴求面へ。

紙ポスターや1画面モニターでは埋もれる案内・キャンペーン・QR導線を、
3連サイネージで「見せる・伝える・行動につなげる」面に変えます。

初期導入パートナー限定
通常 初期398,000円 / 月額39,800円
→ 初期150,000円 / 月額19,800円

ホテル・宿泊施設 20施設限定
その他業種 各10施設限定
```

### 2ページ目: 1画面ではダメな理由

```text
1画面では、情報が“順番待ち”になります。

一般的な1画面サイネージ:
案内 → キャンペーン → QR

QRが表示される前にお客様が通り過ぎてしまうことがあります。

Misell 3連サイネージ:
左: 案内
中央: 訴求
右: QR

案内・キャンペーン・QR誘導を同時に出せます。

1画面は、順番に見せる。
3連は、同時に見せて、すぐ行動につなげる。
```

### 3ページ目: 設置イメージ

実際の提案資料では、説明文よりもモックアップ画像を優先する。

モックアップ条件:

- 16:9のテレビモニターを3台、同じサイズで横並び
- 3台の比率はすべて通常のテレビ比率を守る
- モニター以外の説明文字・価格・比較図・アイコンは不要
- 左画面: 館内案内
- 中央画面: キャンペーン訴求
- 右画面: QR誘導
- ホテルロビー、ジム、温浴、クリニックなどの設置イメージを業種別に作る
- インフォグラフィックではなく、リアルな設置モックアップ画像にする
